🛠️ User Guide – Mobile App

📌 Registration

To register, fill in the fields. If the email is not already registered, you can complete the registration.

🔑 Login

You can log in with your email and password or use Google login.

Google Login

If you log in with Google, a modal with your Google accounts will appear, and you need to select the one you want to use.

Account Linking

If your Google account is already registered, the system will ask if you want to combine the accounts.

📑 Tabs Overview

After logging in, you will see five tabs:

  • Home
  • Courses
  • Classy IA
  • Profile
  • Configuration

🏠 Home

The Home tab shows the roles you can have in the app.

📚 Courses

In the Courses tab, you can see the courses divided into 5 sections:

  • My Courses (your own courses)
  • Public Courses (open courses you can join)
  • Enrolled Courses (courses you are enrolled in or are a professor/assistant)
  • Favorites (courses you favorited)

Create a Course

Tap the plus button at the bottom right to create a course. Complete all the necessary information.

Course Details

If you are a professor, you will see the course details once you enter a course.

📑 Modules

In the Modules section, you will see a list that you can sort as you wish.

Create a Module

Tap the "Create Module" button to create a new module.

Module Details

Each module has its own details. You can see and reorder the resources as you wish.

Add Resource

Tap the plus button to add a resource (image, video, or external link).

View Resource

If it's an image or video, you can view it directly.

External Link

If it's a link, the app will redirect you to the linked page.

📊 Assessment

In the Assessment section, you can see the list of exams or tasks.

Create Assessment

Tap the plus button to create an assessment, including exercises. You can create either open-ended or multiple choice exercises.

Assessment Details

Tap on any assessment to view its details. You can edit or delete it before it opens.

View Exercises

In the "View Exercises" section, you can see how the student will view the assessment.

Export PDF

Tap the top right corner to export the exam as a PDF.

📑 Submissions

In the Submissions section, you can view student submissions and correct them.

Correct Exercise

You can correct each exercise and add comments for each one, as well as provide overall feedback.

💬 Feedback

The Feedback section allows you to:

  • View course feedback.
  • Give feedback to a specific student.
  • View a summary of course feedback generated by Classy.

📊 Course Stats

In this section, you can view the overall performance of the course.

📊 Student Stats

Select a student to view their individual statistics.

Assessment Stats

View statistics for each assessment.

Export Stats

You can also export all the stats as a PDF.

👥 Add Assistant

You can add any user as an assistant to your course.

Review Assistant Activity

Review the activity of your assistants in the course.

Delete Course

As a course creator, you can delete the course if needed.

👩‍🎓 Student View

As a student, you have fewer options than a professor. You can view modules and complete assessments.

Complete Assessment

As a student, you can complete open-ended or multiple-choice assessments.

Give Feedback

As a student, you can provide feedback to the course.

👨‍🏫 Assistant View

As an assistant, you have similar access to a professor, but with some limitations.

  • You cannot delete things, add assistants, or view assistant activity.

🤖 Classy IA

In the Classy IA tab, you can consult things about the app or your courses/exams.

👤 Profile

In the Profile tab, you can view and edit your personal details.

Edit Profile

You can also edit your profile information.

👥 Users

As a professor, you can view all users in the system.

View User Info

Click on a user to view their public information.

My Feedbacks

See all the feedback you have received, along with an AI-generated summary.

⚙️ Configuration

You can log out or configure your notifications in the settings section.
